## **A Simple Boilerplate containing -**

-   Next.js
-   Eslint
-   Prettier
-   CSS Nano
-   Babel
-   Post-CSS
-   Purge CSS
-   TypeScript

> **To Use / Install this boilerplate** -

```
git clone --depth 1 --single-branch https://github.com/sahilbest999/NextJS-simple-boilerplate.git your-project-name
```

**To Run Dev Server** -

```
npm run dev
```

_or_

```
yarn dev
```

---

**To Build** -

```
npm run build
```

_or_

```
yarn build
```

---

**To Build and Export** -

```
npm run export
```

_or_

```
yarn export
```
